# MANUAL DE PROCEDIMIENTOS OPERATIVOS - CLÍNICA SALUD+

## 1. PROCESO DE ADMISIÓN DE PACIENTES

### 1.1. Admisión en Ventanilla
1. El paciente se presenta en la ventanilla de admisión.
2. Presenta documento de identidad y carné de EPS.
3. El auxiliar administrativo verifica la afiliación en el sistema.
4. Se confirma la cita o se agenda una nueva.
5. Se asigna el número de historia clínica.
6. Se imprime la pulsera de identificación.

**Tiempo estimado:** 10 minutos.

### 1.2. Admisión por Urgencias
1. El paciente llega al área de triaje.
2. Enfermera de triaje clasifica la urgencia (Triage I a V).
3. Triage I: Atención inmediata (rojo).
4. Triage II: Atención en < 15 minutos (naranja).
5. Triage III: Atención en < 60 minutos (amarillo).
6. Triage IV: Atención en < 120 minutos (verde).
7. Triage V: Atención en < 240 minutos (azul).

**Procedimiento de triaje:**
- Tomar signos vitales (presión, frecuencia, temperatura).
- Identificar síntomas principales.
- Clasificar según protocolo de Manchester.

## 2. PROCESO DE HOSPITALIZACIÓN

### 2.1. Ingreso a Hospitalización
1. El médico tratante emite la orden de hospitalización.
2. Se asigna habitación según disponibilidad.
3. El paciente o familiar firma el consentimiento informado.
4. Enfermería realiza el ingreso en el sistema.
5. Se entrega kit de bienvenida (pijama, toalla, artículos de aseo).
6. El médico tratante realiza la historia clínica de ingreso.

### 2.2. Visita Médica Diaria
1. El médico tratante realiza visita entre 7:00 am y 9:00 am.
2. Enfermería entrega informe de evolución.
3. Se actualiza el plan de tratamiento.
4. Se agenda exámenes o procedimientos pendientes.

### 2.3. Alta Hospitalaria
1. El médico emite la orden de alta.
2. Se prepara la fórmula médica para entrega al paciente.
3. Enfermería explica las instrucciones post-alta.
4. Se agenda cita de control (7-10 días después).
5. Se facturan los servicios y se gestiona el pago con EPS.
6. El paciente firma el acta de alta.

## 3. PROCESO DE CIRUGÍA PROGRAMADA

### 3.1. Preoperatorio
1. El médico especialista define la necesidad quirúrgica.
2. Se agenda la cirugía (quirófano + anestesiólogo).
3. Se solicitan exámenes preoperatorios (hemograma, coagulación, ECG, RX).
4. El paciente firma el consentimiento informado quirúrgico.
5. Se dan instrucciones preoperatorias (ayuno 8 horas, suspensión de medicamentos).

### 3.2. Día de la Cirugía
1. El paciente ingresa a la clínica 2 horas antes de la cirugía.
2. Se realiza el check-in en la unidad de cirugía.
3. Se coloca la pulsera de identificación y se verifican datos.
4. El anestesiólogo realiza evaluación pre-anestésica.
5. El paciente ingresa a quirófano.
6. Se realiza la cirugía según el procedimiento programado.

### 3.3. Postoperatorio
1. El paciente pasa a la unidad de recuperación (UCPA).
2. Se monitorean signos vitales cada 15-30 minutos.
3. El médico evalua la evolución post-quirúrgica.
4. Se da de alta cuando el paciente está estable.
5. Se entregan instrucciones postoperatorias detalladas.
6. Se agenda cita de control en 7 días.

## 4. PROCESO DE FARMACIA

### 4.1. Dispensación de Medicamentos
1. El médico prescribe el medicamento en la fórmula médica.
2. El paciente se presenta en la farmacia con la fórmula.
3. El farmacéutico verifica la prescripción y la cobertura.
4. Se entrega el medicamento con instrucciones de uso.
5. Se registra en el sistema la entrega y el lote.

### 4.2. Control de Inventario
1. Inventario diario de medicamentos de alto consumo.
2. Revisión de vencimientos cada semana.
3. Pedido automático cuando el stock baja del 20%.
4. Auditoría mensual del inventario.

## 5. PROCESO DE FACTURACIÓN

### 5.1. Facturación a EPS
1. Se registran los procedimientos en el sistema (cada 24h).
2. Se generan los RIPS (Registro Individual de Prestación de Servicios).
3. Se envía la factura electrónica a la EPS.
4. Se realiza el seguimiento de pago (plazo 30 días).
5. Gestión de glosas o devoluciones (15 días hábiles).

### 5.2. Facturación a Particular
1. Se genera la factura al momento del servicio.
2. Se ofrecen opciones de pago (efectivo, tarjeta, transferencia).
3. Se entrega factura física o electrónica.
4. Se registra el pago en el sistema.
5. Se emite el soporte o recibo de pago.

## 6. PROCESO DE QUEJAS Y RECLAMOS

### 6.1. Recepción
1. El paciente se acerca al área de servicio al cliente.
2. Se diligencia el formato de queja o reclamo (PQRS).
3. Se asigna número de radicado.
4. Se entrega constancia de radicación.

### 6.2. Gestión
1. Se analiza la queja y se define responsable.
2. Plazo máximo de respuesta: 15 días hábiles.
3. Se contacta al paciente para dar explicación.
4. Se implementan acciones correctivas si aplica.
